# Senior Backend Engineer - Cloud Infrastructure

**Company:** TechScale Inc. (Use any real company from LinkedIn during demo)
**Location:** Remote (US)
**Salary:** $180,000 - $220,000
**Experience:** 7+ years

## About the Role

We're looking for a Senior Backend Engineer to join our Cloud Infrastructure team. You'll be responsible for building and scaling our microservices architecture serving millions of users daily. This is a hands-on technical leadership role where you'll mentor junior engineers while contributing code.

## Key Responsibilities

- Design and implement scalable backend services using Python and modern frameworks
- Lead migration from monolithic to microservices architecture
- Build and maintain event-driven systems handling high-throughput data streams
- Optimize API performance and database queries for sub-100ms response times
- Implement monitoring, alerting, and observability across all services
- Mentor team members through code reviews and pair programming
- Collaborate with product and frontend teams on API contracts
- Participate in on-call rotation for production systems

## Required Qualifications

- **7+ years of backend software engineering experience**
- **Expert-level Python proficiency** (or Go/Java with willingness to learn Python)
- **Strong experience with FastAPI, Django, or Flask**
- **Production Kubernetes experience** - deploying and managing containerized applications
- **AWS cloud services** - EC2, RDS, S3, Lambda, ECS
- **Microservices architecture** - breaking monoliths into services, service mesh
- **Database expertise** - PostgreSQL, Redis, or similar (both SQL and NoSQL)
- **API design** - RESTful APIs, GraphQL, gRPC
- **Message queues** - Kafka, RabbitMQ, or AWS SQS
- **CI/CD pipelines** - GitHub Actions, Jenkins, or GitLab CI
- Strong understanding of distributed systems and concurrency

## Nice to Have

- Experience with event-driven architectures
- Performance optimization at scale (10M+ requests/day)
- Docker multi-stage builds and optimization
- Infrastructure as Code (Terraform, CloudFormation)
- Monitoring tools (Grafana, Prometheus, Datadog)

- Contributions to open-source projects
